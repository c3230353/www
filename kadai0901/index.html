Box Design Practice

Box 1

This is the first box. It has a light blue background, rounded, corners, and a solid border. This box includes an image below to
enhance the visual appeal of the content. The image should be relevant to the content described and help illustrate the point.

Box 2

This is the second box. It has a light orange background, slightly rounded corners, a solid border, and a shadow. including an
image in this box helps break up the text and rovides a visual example of the content. The use of CSS box shadow enhances the
3D effect of this box.

Box 3

This the third box. It has a light puple background, no rounded corners, and a dashed border. This box also includes an images,
which helps to demonstrate how different style and borders can be applied to create distinct visual sections within a webpage.
